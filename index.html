Smart Store – Sistema de Gerenciamento de Vendas

Projeto pessoal – Python (Flask) + SQLite + Tailwind CSS

Smart Store é um sistema de gerenciamento de vendas simples e eficiente, ideal para pequenas empresas. Ele oferece uma solução completa para registrar vendas e monitorar o desempenho do seu negócio.

✨ Funcionalidades Principais

  • Controle de Vendas: registre produtos, quantidade, preço e custo
  • Armazenamento Híbrido: dados locais e online para acesso centralizado
  • Relatórios Automáticos: melhores produtos, lucro total e desempenho por produto
  • Design Responsivo: acesse pelo celular ou computador
  • Fácil de Implantar: instalação simples em Render ou Railway

🛠️ Tecnologias Utilizadas

Python (Flask), SQLite, Pandas, HTML + Tailwind CSS

🚀 Como Executar

cd smart-store
pip install -r requirements.txt
python app.py
Abra o navegador em http://127.0.0.1:5000/

Projeto Online: smart-store-t54m.onrender.com

GitHub: smart-store

Scanner de Vulnerabilidades Web

Projeto acadêmico – Python

Projeto experimental em Python para detectar vulnerabilidades básicas em sites web, com foco em SQL Injection (SQLi) e Cross-Site Scripting (XSS). O objetivo é auxiliar no aprendizado de segurança web de forma automatizada e educativa.

O scanner recebe uma URL com parâmetros GET e tenta injetar payloads de teste. Ele detecta mensagens de erro de banco de dados (indicando SQLi) e reflexo de código HTML/JavaScript (indicando XSS), gerando um relatório simples no terminal.

Estrutura do projeto: 

web-vuln-scanner/
├── scanner/
│   ├── core.py
│   ├── tests.py
│   └── utils.py
├── main.py
├── requirements.txt
└── README.md

Possíveis melhorias incluem adicionar novas vulnerabilidades, criar relatórios HTML/JSON, suporte a múltiplas URLs e interface gráfica.

GitHub: web-vuln-scanner

Sistema MRP - Fábrica de Bicicletas e Computadores

Projeto acadêmico – PHP + MySQL

Sistema web desenvolvido para planejar a necessidade de materiais (MRP) com base na produção planejada de bicicletas e computadores, garantindo o controle de estoque e evitando a falta de insumos.

✅ Pré-requisitos

  • PHP 8.x
  • MySQL ou MariaDB
  • Navegador web (Google Chrome, Firefox, etc)
  • Servidor local como XAMPP, WAMP ou Laragon

Depois importe o banco no phpMyAdmin, configure o db.php e acesse:

  • Gerenciar estoque: http://localhost/projeto-mrp/src/frontend/estoque.php
  • Calcular MRP: http://localhost/projeto-mrp/src/frontend/mrp.php

📦 Estoque

Permite visualizar componentes, atualizar quantidades e salvar no banco de dados.

📊 Cálculo do MRP

Informe a produção de bicicletas e computadores, clique em "Calcular" e o sistema mostra o necessário de cada item, o que há em estoque e o que precisa ser comprado. 

Produto |Componente|Necessário|Estoque|Comprar
Bicicleta| Rodas     |12|10| 2 ⚠️
Computador|Gabinetes| 3 | 2| 1 ⚠️
