Scanner de Vulnerabilidades Web

Projeto acadêmico – Python

Projeto experimental em Python para detectar vulnerabilidades básicas em sites web, com foco em SQL Injection (SQLi) e Cross-Site Scripting (XSS). O objetivo é auxiliar no aprendizado de segurança web de forma automatizada e educativa.

O scanner recebe uma URL com parâmetros GET e tenta injetar payloads de teste. Ele detecta mensagens de erro de banco de dados (indicando SQLi) e reflexo de código HTML/JavaScript (indicando XSS), gerando um relatório simples no terminal.

Estrutura do projeto: 

web-vuln-scanner/
├── scanner/
│   ├── core.py
│   ├── tests.py
│   └── utils.py
├── main.py
├── requirements.txt
└── README.md

Possíveis melhorias incluem adicionar novas vulnerabilidades, criar relatórios HTML/JSON, suporte a múltiplas URLs e interface gráfica.
